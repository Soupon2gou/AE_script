# AE_script

## ShuffleLayerPositions

### 説明
選択されたレイヤーの位置をシャッフルする.
 -選択されたレイヤーの[x,y]位置配列がランダムでシャッフルされます.

### 導入
Releasesは用意しない予定なのでファイル名を直接クリックして「Download raw file」
から直接ダウンロードしてください.
